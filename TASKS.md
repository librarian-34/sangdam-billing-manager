## 기술 메모

- **데이터 영속성**: localStorage `billing_db` 키에 자동 저장. 새로고침 시 자동 복원.
- **파일 연동**: fileHandle 있으면 변경 시마다 조용히(toast 없이) 파일에도 저장.
- **인쇄 이력 저장 시점**: `window.print()` 직전 이벤트 리스너 등록 → `afterprint` 이벤트 시 저장.
  단, 브라우저 특성상 인쇄 대화상자 취소 시에도 `afterprint`가 발생할 수 있음.
- **캘린더 scope**: `calendar.readonly` → `https://www.googleapis.com/auth/calendar` (읽기+쓰기)로 변경. 기존 사용자 재인증 필요.
- **캘린더 이벤트 삭제**: 그리드 내 이벤트 배지 클릭 → confirm 대화상자 → Google Calendar API로 삭제 → 자동 새로고침.
